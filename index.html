Product Overview

製品情報

当社のクライアントは、最も過酷な製造環境や使用条件においても、高い製造品質と優れた性能を備えた当社の Oリングに信頼を寄せています。すべてのGumlast™およびKratos™の部品は、最新鋭かつ高度に近代化された ISO 9001 認証取得の生産ラインで製造されており、非常に高い生産効率を実現しています。

超高性能エラストマー化合物は、長期的に所有コストを削減するだけでなく、以下の特長を備えています：

  • ✓ フッ素化骨格とナノフィル構造を採用
  • ✓ Interstitial Reinforcement System (IRS)を使用
  • ✓ 最高使用温度275°C (527°F)を超える耐熱性
  • ✓ 金属、難融金属、金属酸化物を一切含まない
  • ✓ 汚染を排除
  • ✓ 既存のハードウェアに対してコスト効率の良い直接交換が可能
  • ✓ FKMとFFKMの用途のギャップを埋める
  • ✓ 設置コストを削減
  • ✓ 定期メンテナンス(PM)サイクルの頻度を向上
  • ✓ FFKMに匹敵する性能を発揮しながら、透過性はFKMに近く、FFKMのような変形が発生しない

Client Support

技術サポート

当社では、Flodynamix製品の導入に関して、エンドユーザーの業界特性や要件に応じたカスタマイズされた技術サポートを提供しています。 シーリング技術は、環境条件や使用される化学物質によって最適な選定が必要となるため、専門の技術スタッフが最適な製品の選定をサポートし、導入後のメンテナンス指導を行います。

  • 製品選定のアドバイス

    用途や使用環境に応じた最適なシーリング材を選定

    プラズマエッチング、CVDプロセス、耐薬品用途などに最適なソリューションを提案

  • 技術的な問題解決

    シールの劣化や故障の原因分析と解決策の提供

    設備のダウンタイム削減をサポートし、生産効率を向上

  • 適切なメンテナンス指導

    長期間の使用を想定した最適な交換スケジュールの提案

    劣化を防ぐための適切な保守・管理方法を指導
